# Clínica Dental Premium Uchuya

Aplicación web profesional construida con **Next.js 15**, **Tailwind CSS v4** y **Framer Motion**. Diseñada para transmitir confianza, lujo y captar clientes potenciales de manera efectiva.

## 🚀 Tecnologías
- **Framework**: Next.js 15 (App Router)
- **Estilos**: Tailwind CSS v4 (Configuración CSS-first)
- **Animaciones**: Framer Motion
- **Formularios**: React Hook Form + Zod
- **Iconos**: Lucide React
- **Imágenes**: Optimización nativa con `next/image`

## 🛠️ Instalación y Ejecución Local

1.  **Clonar/Descargar el proyecto**
2.  **Instalar dependencias**:
    ```bash
    npm install
    ```
3.  **Ejecutar en desarrollo**:
    ```bash
    npm run dev
    ```
    La aplicación estará disponible en `http://localhost:3000`.

## 🌐 Despliegue en Vercel

Esta aplicación está lista para ser desplegada en Vercel con un solo clic:
1. Sube el código a un repositorio de GitHub/GitLab/Bitbucket.
2. Importa el proyecto en [Vercel](https://vercel.com).
3. Vercel detectará automáticamente Next.js y realizará el despliegue.

## 📱 WhatsApp Inteligente
El botón flotante permite:
- Seleccionar la sede (Tingo María o Aucayacu).
- Seleccionar el tratamiento de interés.
- Generar un mensaje personalizado que redirige al número correcto de la sede elegida.

## 🎨 Diseño Premium

- **Tipografía**: Outfit (Headings) e Inter (Body).
